## Installation

```bash
$ npm install
```

## Run project
```bash
#Run docker
$ npm run up
#Stop docker
$ npm run down
#Run app
$ npm run start:dev
#Allow migrations
$ npm run db:migrate
#Drop DB
$ npm run db:drop
```
